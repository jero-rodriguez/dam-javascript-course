Transformación a Mayúsculas

Escribe tu nombre en minúsculas y pulsa fuera del cuadro o usa el tabulador.

El cambio ocurre al perder el foco (onblur).

Lo que aprendemos en este vídeo:
- onblur: Evento que se ejecuta cuando un elemento deja de tener el foco del usuario.
- toLocaleUpperCase(): Método de cadena que convierte el texto a mayúsculas respetando la configuración regional (acentos, letra ñ, etc.).
- Manipulación de valor: Cómo leer el contenido de un input (`.value`) y reescribirlo modificado.
- Interactividad: Mejorar la experiencia del usuario automatizando el formato de los datos.
