Validación: Fecha de Hoy

Selecciona una fecha para comprobar si es válida (no anterior a hoy).

La fecha no puede ser anterior al día de hoy.
Lo que aprendemos en este vídeo:
- new Date(): Obtiene la fecha y hora completa del sistema actual.
- toISOString(): Convierte el objeto fecha a una cadena de texto estándar ISO.
- slice(0, 10): Recorta los primeros 10 caracteres para obtener el formato AAAA-MM-DD.
- Comparación de Strings: Al usar el formato año-mes-día, podemos comparar fechas usando operadores matemáticos.
