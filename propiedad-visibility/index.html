Control de Visibilidad con JS

Elige tu destino. Si seleccionas "España", aparecerá el selector de provincias.

Lo que aprendemos en este vídeo:
- visibility: hidden: Oculta el elemento pero mantiene el espacio que ocupa en el diseño.
- visibility: visible: Hace que el elemento vuelva a ser perceptible.
- onchange: Evento que se dispara cuando el usuario cambia la selección de un menú desplegable.
- control.style.visibility: Forma de acceder y modificar esta propiedad CSS desde JavaScript.
