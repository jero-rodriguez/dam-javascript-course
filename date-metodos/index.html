Métodos del Objeto Date

Selecciona una fecha para descomponerla usando métodos de JavaScript.

Lo que aprendemos en este vídeo:
- new Date(string): Crea un objeto de fecha a partir de una cadena de texto.
- getDate(): Extrae el número del día del mes (1-31).
- getDay(): Devuelve el día de la semana (donde 0 es domingo).
- getMonth(): Devuelve el mes del año (donde 0 es enero).
- getFullYear(): Obtiene el año con cuatro dígitos.
