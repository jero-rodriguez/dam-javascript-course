Registro de Usuario

Puntos clave de la validación:
- onsubmit="return funcion()": Si la función devuelve `false`, el formulario NO se envía.
- .value.length: Se usa para contar caracteres (útil para contraseñas).
- .focus(): Coloca el cursor automáticamente en el campo que tiene el error.
- Validación de Select: Comprobamos si el valor sigue siendo el inicial ("ninguna").
