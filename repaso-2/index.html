Manejo de Imágenes e Inputs



Conceptos de Manejo del DOM:
- document.getElementById(): "Agarra" un elemento por su ID.
- innerHTML: Cambia el contenido dentro de una etiqueta.
- style.propiedad: Modifica el CSS (ej: .style.color).
- src y value: Cambia atributos directos de imágenes e inputs.
- className: Aplica una clase CSS completa de golpe.
